# Contributing

English: [CONTRIBUTING.md](CONTRIBUTING.md)

## Vor einem Pull Request

- Änderungen sollten sich auf das Docker-Image, das Compose-Setup oder den Release-Workflow konzentrieren.
- Aktualisiere `README.md`, `README.DE.md` und die Changelog-Dateien bei nutzerrelevanten Änderungen.
- Prüfe Build und Laufzeitverhalten vor der Veröffentlichung.
